What counts as an event?

Events are the unit of measure for tracking GlitchTip usage. Your plan's monthly limit is shared across all features.

breaking_news Issues
Each occurrence of an error or exception reported to GlitchTip
share_eta Uptime
Each health check of your app, whether polled by GlitchTip or pushed from your app
avg_pace Performance
Each transaction report for a tracked activity on your app
text_snippet Logs
Each log entry counts as 1/10 of an event — 10 log entries equal 1 event
rocket_launch Releases
Each megabyte of file storage is counted as an event
